# SteelBench (anonymous code release)

This is the anonymized code repository for the NeurIPS 2026 Datasets &
Benchmarks submission **SteelBench: Evaluating Vision-Language Models in
Real-World Industrial Environments**.

The dataset is hosted on Hugging Face:
  https://huggingface.co/datasets/steelbench/SteelBench

For reproduction instructions see [`REPRODUCE.md`](REPRODUCE.md).
For the audit protocol contribution see [`audit/README.md`](audit/README.md).
